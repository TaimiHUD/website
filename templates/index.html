TaimiHUD is an MIT-licensed cross-platform Nexus and ArcDPS Guild Wars 2 addon written in Rust that provides functionality generally only seen on Windows, provided by TacO, BlishHUD, etc.

It was made with the intention of providing a similar feature set to those of us on Linux and to those of us who did not want an overlay window, but something rendering within the game.

Achievement API integration and marker interaction is now in alpha! If you would like to help test and preview those features, please see our testing channel on Discord for more information.
