TaimiHUD draws markers, trails, and timers right inside Guild Wars 2 — no overlay window, on Linux as well as Windows.

Already running Nexus? TaimiHUD is in its addon library. New to addons? The install guide walks you through the whole setup.

TaimiHUD is an MIT-licensed cross-platform Nexus and ArcDPS addon written in Rust that provides functionality generally only seen on Windows, provided by TacO, BlishHUD, etc.

Achievement filters, interactions, and GogglesV2 are now in testing

If you would like to help test and preview new features, you can try it out!
